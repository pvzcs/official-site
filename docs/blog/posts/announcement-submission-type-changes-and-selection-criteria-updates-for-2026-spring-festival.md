# 关于从2026新春会起变更稿件类型和提升部分类型稿件的入选门槛的说明

我们已决定，从2026新春会起整合当前部分稿件类型并提升部分类型稿件的入选门槛，具体细则如下：

1. 原自制关卡、自制植物、自制僵尸和游戏的衍生修改和创作-其他四类将全部整合至自制关卡类目，并取消一级分类。
2. 对于自制关卡稿件，我们将变更：
    1. 自制关卡类稿件将完全剔除对其本身的技术力考量，转而仅从观看体验维度上对自制关卡进行审核
    2. 在审核阶段中，我们将特别关注呈现方式、接受门槛、引导方式和创新程度四个维度
3. 对于燃剪混剪类稿件，我们将变更：
    1. 接受部分燃剪混剪UP主加入审稿
    2. 针对存在内容同质化的燃剪混剪稿件酌情扣分降档，或不予入选
